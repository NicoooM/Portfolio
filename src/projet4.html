Refonte de site web

In Git

Missions

Durée

2 semaines

Le projet design intéractif est le premier projet web que j’ai réalisé durant mon DUT. Le concept est simple : faire la refonte d’un site parmi plusieurs existants. C’est donc dans ce projet que j’ai réellement découvert le monde du web.

J’ai donc pu, durant ce projet, passer par toutes les étapes de refonte d’un site. Nous avons commencé par faire le zoning, le wireframe puis la maquette. Nous avons ensuite réalisé le site en lui même, en HTML, CSS et un peu de JS. Je me suis personnellement occupé de la partie de Franck le garagiste et des derniers avis clients.
