Sing Karaoké

In Git

Missions

  • Front avec React
  • Manipulation de données

Durée

2 semaines

Sing Karaoké est un site web mobile first réalisé lors de ma formation. Le but était de créer un karaoké, et donc de faire apparaître les paroles en fonction de l'avancement de la musique. Nous avons utilisé des fichiers LRC, qui sont des fichiers qui associent des paroles à un temps.
