Projet 02

Création d’un site de A à Z

In Git

Missions

  • UX Design
  • UI Design
  • Intégration

Durée

2.5 semaines

Le projet main dans la main est le second gros projet du DUT MMI concernant le web. Nous devions réaliser un site en partant de zéro, de l’idéation au développement. Le thème était le covid-19, nous avons donc décidé de faire un site vitrine d’un site de location de matériel pour les étudiants (site de location fictif bien entendu).

En ce qui concerne les technologies utilisées, nous nous sommes servis de Jekyll pour travailler plus facilement en équipe. Nous utilisons aussi le préprocesseur Sass, en plus des langages du web de base. Le développement du site a été fait en mobile first, et par manque de temps et de connaissances à ce moment-là, nous n’avons pas pu faire complètement la version ordinateur. A voir donc de préférence sur mobile.
